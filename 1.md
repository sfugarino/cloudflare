1. Install open-iscsi
sudo apt install open-iscsi

2. Clone Github repository
git clone https://github.com/jetsonhacks/jetson-orin-kernel-builder.git

3. Change directory
cd jetson-orin-kernel-builder

4. Get Kernel sources

./scripts/get_kernel_sources.sh

5. Edit build configuration
./scripts/edit_config_cli.sh

Device Drivers > SCSI device support > SCSI low-level drivers > iSCSI Initiator over TCP/IP 

6. Make kernel modules 
./scripts/make_kernel_modules.sh

7. Copy files to modules directory
sudo cp /usr/src/kernel/kernel-jammy-src/drivers/scsi/*.ko /lib/modules/$(uname -r)/scsi

8. Run depmod
sudo depmod -a 

9. Load libiscsi module
sudo modprobe libiscsi

10. Load iscsi_tcp module
sudo modprobe iscsi_tcp

11. Restart iscsid service
sudo systemctl restart iscsid

12. Make permenant
   a. sudo touch /etc/modules-load.d/iscsi.conf
   b. echo "libiscsi" >> /etc/modules-load.d/iscsi.conf
   c. echo "iscsi_tcp" >> /etc/modules-load.d/iscsi.conf
